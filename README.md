# Codenames UA (Expo / React Native)

Простий iPhone-застосунок гри **Codenames** з локальною грою для 2 команд (червоні/сині), ролями капітана та оперативників.

## Що реалізовано

- Поле 5x5 зі словами
- Випадкова генерація ключа: 9/8 агентів, 7 мирних, 1 вбивця
- Хід по черзі для синіх/червоних
- Підказка капітана: **1 слово + число**
- Перевірка базових заборон для підказки (не можна використовувати слово/частину слова з поля)
- Вгадування оперативників з правилом **+1**
- Завершення гри при відкритті всіх агентів команди або вбивці
- Режим "Капітан" (показує ключ) / режим "Оперативники" (прихований ключ)

## Запуск

```bash
npm install
npm run ios
```

> Для локального запуску iOS потрібен macOS + Xcode.

## Підготовка до App Store

У `app.json` вже додані базові iOS параметри:

- `expo.name`: `Codenames UA`
- `expo.slug`: `codenames-ua`
- `expo.ios.bundleIdentifier`: `com.toniartemenko.codenamesua`

Перед публікацією потрібно:

1. Оновити `bundleIdentifier` на унікальний вашого Apple Developer акаунта (за потреби)
2. Додати фінальні іконки, splash, метадані та скріншоти
3. Збільшити `version` і налаштувати збірку через EAS/Transporter
